<strong 태그>

stong태그: 글씨를 두껍게 해주어서 강조한다.
ex) <strong> 고상진 </strong> --> 고상진

<u태그>

u태그: 글씨 아래 밑줄을 쳐준다
ex) <u> 고상진 </u> --> 고상진

<h태그>

h태그: 제목 형식을 만들어 준다.
h태그는 자체적인 줄 바꿈이 존재한다.
h태그의 형식은 <hn> "내용" <hn> (n은 1~6까지의 자연수)

다음은 h태그의 예시이다.

ex)

고상진(h1)

고상진(h2)

고상진(h3)

<br태그>

br태그: 줄넘김의 태그이다.
ex)
고상진1 <br> 고상진2
--->

고상진1
고상진2

<p태그>

p태그: 단락넘김의 태그이다.
ex)
고상진1 <p>고상진2</p>
-->

고상진1

고상진2

<img태그>

img태그: 이미지(사진)을 삽입하는 태그이다.
이미지를 프로젝트 폴더에 다운 --> 코딩 화면에서 우클릭 후 rename하여 새 이름으로 저장 -->
<img src="(rename한 이름).jpg" width="퍼센트 값 or 숫자">를 입력하여 사진 삽입한다. (width는 사진의 크기를 설정해줌+width, src는 속성이라고 부르고 순서는 상관없다.)
ex) <img src="초등.jpg" width="100%">
-->

<부모자식의 관계li태그, ul태그(unordered list)>

ex)
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

-->

  • HTML
  • CSS
  • JavaScript
ul태그 사용하고 ul태그에 포함되는 li태그들을 tab키로 안으로 들여쓰면 출력도 안으로 들여써짐
* 팁: ctrl을 누른상태로 클릭을하면 여러군데를 클릭할 수 있음

<자동으로 숫자로 리스트화하는 태그 ol태그(ordered list)>

ex)
<ol>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ol>

-->

  1. HTML
  2. CSS
  3. JavaScript
